# To-Do List App

**A simple to-do list web application built to practice the fundamentals of HTML, CSS, JavaScript, and responsive design.**  
Users can add, complete, and remove tasks using an intuitive UI and interact with the list on any device.

## About

This project was designed to practice core frontend skills by building a fully interactive task list. The app's features include:

-  Adding new tasks  
-  Marking tasks as complete  
-  Deleting tasks  
-  Responsive layout for mobile and desktop  
-  JavaScript DOM manipulation and event handling

**Installation & Local Setup**

To run this project on your computer:

Clone the repository

git clone https://github.com/ryanhollingsworth123/To-Do-List-App.git

Navigate to the project folder

cd To-Do-List-App
Open the app in your browser

Double-click index.html, or

Right-click and choose “Open with Live Server” (VS Code) or similar.

**How It Works**

Type a task into the input field and press Enter or click Add.

The task appears in the list below.

Click on a task to mark it complete.

Click the delete button next to a task to remove it.

The app is fully responsive and works across different screen sizes.

**Goals & Learning Outcomes**

This project practices:
